# WoW Retail – Fraktionen nach Expansion (bereinigte Matrix)

Diese Datei ist für den **Addon-Abgleich** gedacht und listet die **spielerrelevanten Ruf-/Renown-Fraktionen** nach der Expansion, in der sie **eingeführt** wurden.

## Scope
Enthalten sind:
- normale Ruf-Fraktionen
- Major-Factions / Renown-Tracks
- Raid-/Patch-/Kampagnen-Fraktionen
- klar erkennbare Subfactions, wenn sie im Spiel separat als Fortschritt/Faction auftauchen

Nicht als eigener Hauptblock gepflegt sind:
- reine **Freundschafts-/Individual-NPC-Reps**
- **Guild Reputation**
- rein technische / eventinterne / vorübergehende nicht standardisierte Systeme

## Spalten
- **Fraktion** = Name der Fraktion
- **Seite** = Allianz / Horde / Neutral / Hostile
- **Typ** = grobe Einordnung für Addon-Mapping
- **Hinweis** = kurze Einordnung
